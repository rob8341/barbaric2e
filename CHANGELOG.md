# Changelog

## [1.0.0] - Initial Release

### Added
- Character sheet with 4 tabs (Main, Equipment, Sorcery, Bio)
- NPC sheet with stats, skills, weapons, armor, and abilities
- 6 Skills: Combat, Craft, Lore, Physical, Social, Stealth
- 5 Ranks: Experienced, Expert, Master, Grandmaster, Legend
- Skill rolls with 2d6 + skill + modifier - fatigue
- Advantage rolls (3d6 drop lowest)
- Critical success (double 6s) and critical failure (double 1s) detection
- Weapon damage rolls
- Triage rolls (2d6 - wounds)
- Combat initiative (2d6 + combat skill)
- Equipment management with equip toggle
- Coin tracking (Gold, Silver, Copper)
- Wounds and Fatigue tracking
- Combat moves (levels 2-5)
- Spells with range and description
- Archetype as an item
- NPC abilities with name and description
- Sword and sorcery dark theme
- Themed chat messages
