# Barbaric! 2e - Foundry VTT System

A sword and sorcery 2d6-based tabletop RPG system for Foundry VTT v13.

## Features

### Characters
- **6 Skills**: Combat, Craft, Lore, Physical, Social, Stealth
- **5 Ranks**: Experienced, Expert, Master, Grandmaster, Legend (5 XP to advance)
- **Stats**: Stamina, Defense, Armor, Hero Points, Wounds, Fatigue
- **Archetype**: Short character description as an item
- **Combat Moves**: Levels 2-5
- **Equipment**: Weapons, Armor, General Equipment with equip toggle
- **Sorcery**: Spell tracking
- **Bio**: Appearance, Personality, Motivation, Notes
- **Coins**: Gold, Silver, Copper tracking

### NPCs
- Short description
- Stamina, Defense, Movement, Armor
- All 6 skills with roll buttons
- Weapons with damage rolls
- Armor items
- Special NPC Abilities

### Dice Rolling
- **Skill Rolls**: 2d6 + Skill + Modifier - Fatigue
- **Advantage**: Optional 3d6 drop lowest
- **Critical Success**: Double 6s
- **Critical Failure**: Double 1s
- **Damage Rolls**: Roll weapon damage formula
- **Triage Rolls**: 2d6 - Wounds + Modifier
- **Initiative**: 2d6 + Combat skill

## Installation

### Manual Installation
1. Download the latest release zip file
2. Extract to your Foundry VTT `Data/systems/` directory
3. Restart Foundry VTT

### Manifest URL
```
https://github.com/YOUR_USERNAME/barbaric2e/releases/latest/download/system.json
```
